CONHECENDO O HTML EO CSS

  • HTML

HTML É a linguagem usada para estruturar e exibir o contúdo de páginas na web.Em um site, o Html é o que organiza elementos como textos, imagens, links,formulários, entre outroos. Ele funciona através de "Tags" (ou "Etiquetas"), que são instruções em angulos como (html, head, body) usadas para definir o que cada parte do conteúdo representa.

CSS É a linguagem usada para estilizar páginas HTML, ou seja, para definir a aparencia visual de cada elemento (cores, fontes, espaçamento, etc.). Ele funciona aplicando "Propriedades" aoos elementos Html neste caso as Tags.
